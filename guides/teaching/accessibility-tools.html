This comprehensive guide covers the accessibility features available on Chrome OS to support students and staff with diverse learning needs. All features are built-in and free to use.

Accessibility on Chromebooks

Chrome OS includes robust accessibility features designed to make technology accessible to everyone. These tools support users with:

  • Vision needs - Screen readers, magnification, high contrast, and more
  • Hearing needs - Captions, visual notifications, and mono audio
  • Motor needs - Voice control, switch access, and on-screen keyboard
  • Cognitive needs - Text-to-speech, dictation, and focus tools

Accessing Accessibility Settings

1

Open Quick Settings

Click the time in the bottom-right corner of your screen to open Quick Settings.

2

Access Accessibility Menu

Click the Settings gear icon, then click Accessibility in the left sidebar.

Shortcut: Press Ctrl + Alt + Z to toggle the accessibility menu quickly.

ChromeVox Screen Reader

1

Enable ChromeVox

To turn on ChromeVox:

  • Go to Settings > Accessibility > Text-to-Speech
  • Toggle ChromeVox on
  • Keyboard shortcut: Press Ctrl + Alt + Z

ChromeVox will speak all text, buttons, and navigation elements aloud.

2

ChromeVox Basic Commands

Common keyboard shortcuts when ChromeVox is enabled:

  • Search + Space: Click/activate element
  • Search + Left/Right Arrow: Navigate by character
  • Search + Ctrl + Left/Right Arrow: Navigate by word
  • Search + H: Navigate by heading
  • Search + L: Navigate by link
  • Search + T: Navigate by table

Note: The "Search" key is the magnifying glass key on your Chromebook keyboard.

Screen Magnifier

1

Enable Magnifier

To turn on screen magnification:

  • Go to Settings > Accessibility > Display and magnification
  • Toggle Full-screen magnifier or Docked magnifier
  • Keyboard shortcut: Press Ctrl + Search + M
2

Using the Magnifier

Control magnification:

  • Ctrl + Alt + Brightness Up/Down: Increase/decrease zoom level
  • Move mouse: Pan around magnified area
  • Full-screen mode: Entire screen zooms in
  • Docked mode: Top portion of screen shows magnified view

High Contrast Mode

Improve text visibility with high contrast:

  • Go to Settings > Accessibility > Display and magnification
  • Toggle High contrast mode
  • Text becomes black on white (or white on black in dark mode)
  • Removes background images and reduces visual complexity

Additional Vision Features

  • Large cursor: Increase cursor size in Display settings
  • Cursor highlighting: Add a colored ring around cursor
  • Color filters: Adjust colors for color blindness
  • Dark theme: Reduce brightness and eye strain

Live Captions

1

Enable Live Captions

Turn on automatic captions:

  • Go to Settings > Accessibility > Audio and captions
  • Toggle Live Caption
  • Keyboard shortcut: Press Search + C

Captions will appear at the bottom of your screen for all audio.

2

Customize Caption Appearance

Adjust caption display:

  • Click caption box to move it around screen
  • Adjust text size in caption settings
  • Change caption background color and opacity

Visual Notifications

Get visual alerts instead of audio notifications:

  • Go to Settings > Accessibility > Audio and captions
  • Toggle Flash screen for notifications
  • Screen will flash briefly when alerts occur

Mono Audio

Combine stereo channels for single-ear listening:

  • Go to Settings > Accessibility > Audio and captions
  • Toggle Mono audio
  • Left and right channels play the same audio

Voice Control (Dictation)

1

Enable Dictation

Use voice to type text:

  • Go to Settings > Accessibility > Keyboard and text input
  • Toggle Dictation
  • Keyboard shortcut: Press Search + D

A microphone icon appears when dictation is active in text fields.

2

Using Voice Commands

Speak naturally to type. Voice commands include:

  • "Period" - Insert period
  • "Comma" - Insert comma
  • "New line" - Start new line
  • "Delete" - Remove last word
  • "Select all" - Highlight all text

On-Screen Keyboard

Display virtual keyboard on screen:

  • Go to Settings > Accessibility > Keyboard and text input
  • Toggle On-screen keyboard
  • Keyboard appears at bottom of screen when text field is active
  • Click keys with mouse or touchpad to type

Switch Access

Navigate using external switches or designated keys:

  • Go to Settings > Accessibility > Keyboard and text input
  • Toggle Switch access
  • Configure switches (keyboard keys or USB switches)
  • Set scanning speed and method (automatic or manual)
  • Items on screen highlight in sequence; activate with switch

Additional Motor Features

  • Sticky keys: Press modifier keys one at a time instead of simultaneously
  • Slow keys: Add delay before key press registers
  • Bounce keys: Ignore repeated key presses
  • Auto-click: Automatically click when cursor stops moving

Select-to-Speak

1

Enable Select-to-Speak

Read selected text aloud:

  • Go to Settings > Accessibility > Text-to-Speech
  • Toggle Select-to-speak
  • A speaker icon appears in the status area
2

Using Select-to-Speak

To have text read aloud:

  • Click the speaker icon in status area
  • Click and drag to select text, or click individual items
  • Selected text will be read aloud
  • Control playback speed and voice in settings

Additional Cognitive Support

  • Reading mode: Simplify web pages for easier reading (available in Chrome)
  • Reduced animations: Minimize motion and transitions
  • Text highlighting: Available with Select-to-Speak to follow along visually
  • Word prediction: Suggest completions while typing

Quick Accessibility Shortcuts

Memorize these keyboard shortcuts for fast access:

  • Ctrl + Alt + Z: Toggle accessibility menu
  • Search + C: Toggle Live Captions
  • Search + D: Toggle Dictation
  • Ctrl + Search + M: Toggle Screen Magnifier
  • Ctrl + Search + H: Toggle High Contrast
  • Search + Shift + S: Activate Select-to-Speak
üí° Customization Tip

All accessibility features can be customized extensively in Settings. Experiment with different settings to find what works best for your specific needs.
